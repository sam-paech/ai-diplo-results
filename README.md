# AI Diplomacy Results Reproduction Materials

https://arxiv.org/abs/2508.07485

This repository contains materials to reproduce the primary results in the Democratizing Dimplomacy paper.

Download the data from here: https://drive.google.com/file/d/1SPC29tzuVwdjRhWFpgPxZRe50HB8mDpN/view?usp=drive_link

And decompress to data/ in this repo dir. The notebooks can then be run and will reference the results files in data/. The notebooks in viz/ will produce the charts found in the paper.

The AI Diplomacy harness commands to produce the results files for the main paper experiments are found in the respectively named .txt files in viz/. To run these commands, first install the harness: https://github.com/GoodStartLabs/AI_Diplomacy
